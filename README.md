## ✨ Özellikler

- 🧠 ChatGPT entegrasyonlu akıllı quizler
- 📱 Tam PWA desteği (Ana ekrana eklenebilir)
- 📊 Seviye analiz sistemi
- 🗂️ Kelime flashcard'ları
- ⚡ Gerçek zamanlı ilerleme takibi
